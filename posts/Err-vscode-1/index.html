  • 원인: config 파일에 대한 경로 에러

중간에 보면 config 파일에 대한 경로가 처음에는 \\으로 시작하다가, /으로 이어진다. 누가봐도 삐꾸같은 상황.

  • 해결: Settings > Extensions > Remote - SSH > Remote.SSH: Config File 에서 아래와 같이 직접 설정해줘야 함.
1

C:\\Users\\USER_NAME\\.ssh\\config
